<b> Python version: 3.11 <br />
Django version: 4.1.7 </b>

## Migrate & Run
```sh
$ run.sh
```

## Run server

with Makefile (Unix):
```sh
$ make run
```

or 

```sh
$ pip manage.py runserver
```

## Requirements

```sh
Python
Django
```
